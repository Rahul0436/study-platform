Introduction To Java

What is Java ?

  • Java is a platform independent object-oriented programming language .

Why java was invented ?

  • Java was invented for achiving platform independent for developing internet based business application .
  • Java was invented to develop advanced software for a wide variety of network devices and embedded systems .

Who did invent java ?

  • Java was invented by Mr. James Gosling in Sun Microsystem in the year 1991 for developing internet applications by achieving platform independent and security .
  • It is released in the year 1995 . The version is released for only testing purpose .
  • The first version for project developement is released in the year 1996 as java 1.0
  • The latest version of java is 22 , it is released on 19 march 2024 .
  • Oracle acquired the Java programming language when it purchased Sun Microsystems in 2010 .

What are the applications of Java programming ?

    In the computer world all software or projects are divided into two types

  • Stand-alone applications
  • Internet applications

Stand-alone application

  • An application that is installed in one computer and it it can not be accessed from other computer via internet , if it can be accessed only within same computer inn which it is installed is called stand-alone application .
  • A stand-alone application is a single user and single computer application .
  • For example : Notepad , Calculator , Ms-office , Mobile apps ....etc .

Internet Application

  • An application that is installed in one computer and if it can be accessed from other computer via internet is called as internet applications.
  • An internet application is a multi user and multi computer application .
  • For example : Youtube , Instagram , Facebook , Amazon , Zoom ....etc .

What is platform ?

  • A platform in an environment in which programs are loaded , executed and generate output .
  • A platform can be software only or platform can be hardware or both .

    There are two types of platform

  • Platform Dependent
  • Platform Independent

Platform Dependent

  • A program's compiled code of one operating system if it can not be executed in different operating system we called it as platform dependent .
  • The programming language using whihc we develop this program is called platform dependent language .
  • For example : C , C++ .

Platform Independent

  • A program's compiled code of one operating system if it can be executed in different operating system we called it as platform idependent .
  • The programming language using which we develop this program is called platform independent language .
  • For example : Java, Python, C#, and JavaScript

What is JDK ?

JDK stands for java developement kit , it internally contains JRE and JVM . The main job of jdk is used to provide all the tools to work with java language .

Why is JVM not platform independent?

JVM is platform dependent because it takes java byte code and generates byte code for the current operating system. So Java software is platform dependent but Java language is platform independent because different operating system have different JVMs

What is JVM ?

JVM stands for java virtual machine it is the software in the form of interpreter written in c language through which we can execute java program .

Java is a platform independent or not ?

Yes, Java is a platform-independent programming language.because it uses a "Write Once, Run Anywhere" approach
