JDK Installation

  • Open Browser
  • Search JDK latest version download
  • Click java download
  • Click windows
  • Click jdk.exe file link
  • Downloading will be started after few second
  • After downloading click on JDK.exe file , then installation will be started very shortly .
  • Click Yes ....click next .....click change
  • Change the path c:/jdk-22 (remove program file\java\)
  • Click ok then next and close .
  • Installation is completed in c:\jdk-22 folder .
